最小二乘法 least squares method

通过最小化误差的平方和,来寻找和观测数据匹配的最佳函数

\[\varepsilon = \sum_{i=1}^{m}(y_i - \hat{y}_i)\]

它的矩阵形式为

\[min|| XB -Y ||^2_2\]

在向量空间中找到一个系数矩阵 B, 使自变量矩阵 X 与 B 的乘积与因变量 Y 之间的欧氏距离的平方最小。
