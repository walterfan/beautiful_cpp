Java NIO

IO 方式分为同步和异步 IO ，或分为 阻塞和非阻塞 IO。

同步是指用户空间的线程是主动发起 IO 请求的一方，内核空间是 被动接受方 异步是指系统内核主动发起 IO 请求，用户空间的线程是被动接收方

阻塞 IO 是指需要内核 IO 操作完成后，才返回到用户空间执行用户的操作，这里的阻塞是指用户空间程序的执行状态。
